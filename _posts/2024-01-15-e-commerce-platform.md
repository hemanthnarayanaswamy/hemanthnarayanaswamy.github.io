## Overview

- Built storefront + admin console with real-time order management.
- Implemented Stripe Checkout, inventory sync, and fulfillment webhook automation.

## Architecture Snapshot

- **Frontend:** React + Chakra UI with Vite bundling.
- **API:** Node.js/Express, JWT auth, and role-based policy gates.
- **Data:** MongoDB (catalog), Redis (sessions), Stripe for payment events.

## Highlights

1. Pluggable discount engine (rule-based).
2. Customer notifications via email/SMS webhooks.
3. Admin dashboards for revenue + funnel metrics.
